# StarCore MedLink 功能说明

## 概述

StarCore MedLink 是您的个人健康 AI 伙伴。它能记住您的完整健康历史，在任何医学咨询场景下提供结合个人情况的个性化回答。

## 核心功能

### 1. 健康咨询对话
- 基于您的个人健康档案（病史、用药、过敏史等）提供个性化建议
- 结合医学文献知识库回答专业问题
- 联网搜索获取最新医学资讯

### 2. 医学影像分析
- **脑肿瘤 MRI 分析**：上传脑部 MRI 图像，AI 自动检测并分割肿瘤区域
- **胸部 X 光分析**：上传胸部 X 光片，判断是否呈现 COVID-19 特征
- **皮肤病变分析**：上传皮肤照片，分割并分类病变区域

### 3. 个人健康档案管理
- 记录慢性病、用药清单、过敏史
- 追踪血压、血糖、体重等健康指标
- 查看历史趋势和健康摘要

### 4. 语音交互
- 支持语音输入（浏览器录音 → 文字转写）
- 支持语音输出（文字 → 语音播报）

## 使用场景

| 场景 | 说明 |
|------|------|
| 头痛咨询 | 系统会结合您的血压记录、用药情况给出个性化建议 |
| 药品查询 | 对照您的当前用药清单，检查药物相互作用 |
| 体检报告解读 | 上传或描述检验报告，获取通俗解读 |
| 影像初筛 | 上传医学影像，获取 AI 初步分析结果 |
| 健康追踪 | 每日记录血压、血糖等指标，追踪变化趋势 |
