# 使用指南

## 快速开始

### 1. 环境配置
1. 复制 `.env.example` 为 `.env`，填入必要的 API Key
2. 必需：`LLM_API_KEY`（DashScope 或 OpenAI 兼容 API）
3. 推荐：`TAVILY_API_KEY`（联网搜索）、`ELEVEN_LABS_API_KEY`（语音功能）

### 2. 启动系统
```bash
# 安装依赖
pip install -r requirements.txt

# 启动后端服务
python app.py

# 访问 Web 界面
# 浏览器打开 http://localhost:8080
```

### 3. 摄入医学文献（可选）
```bash
# 摄入单个文件
python ingest_rag_data.py --file /path/to/document.pdf

# 摄入整个目录
python ingest_rag_data.py --dir /path/to/documents/
```

## 界面说明

### Dashboard（首页）
- 快速问答入口
- 显示最近健康摘要
- 系统状态概览

### ChatPage（对话页）
- 沉浸式聊天界面
- 支持文字输入、图片上传、语音输入
- 左侧菜单可切换不同功能

### 健康档案
- 填写个人健康信息
- 记录每日健康日志
- 查看历史趋势

## 对话技巧

1. **提供上下文**：描述症状时尽量详细（何时开始、严重程度、伴随症状）
2. **利用个人记忆**：系统会结合您的健康档案给出个性化回答
3. **追问**：对 AI 回答不满意时可以追问细节
4. **上传影像**：有医学影像时直接上传，AI 能给出初步分析
5. **要求来源**：可以要求 AI 提供信息来源或参考文献
