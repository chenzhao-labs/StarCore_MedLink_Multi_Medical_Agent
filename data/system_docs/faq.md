# 常见问题 (FAQ)

## 影像分析怎么用？

1. 在聊天页面点击上传按钮（📎图标）
2. 选择您要分析的医学影像文件（支持 PNG、JPG、JPEG 格式）
3. 可选：附带文字描述您的问题
4. 系统会自动识别影像类型并调用对应的 AI 模型分析
5. 分析结果会显示在对话中

支持的影像类型：
- 脑部 MRI（肿瘤检测与分割）
- 胸部 X 光（COVID-19 判断）
- 皮肤病变照片（良恶性分类与分割）

## 如何记录健康数据？

1. 在健康档案页面填写您的个人基本信息（年龄、性别、身高、体重等）
2. 添加慢性病、用药信息、过敏史
3. 通过健康日志功能每日记录血压、血糖、体重等指标
4. 系统会自动生成健康摘要和趋势

## 我的数据安全吗？

- 所有健康数据存储在本地 SQLite 数据库中
- 对话历史基于会话隔离，不同用户之间数据隔离
- 文件上传处理后自动删除临时文件
- 语音数据即时处理后删除

但请注意：
- 当前版本不提供端到端加密
- 建议不要在对话中分享身份证号、社保号等敏感身份信息
- 系统调用第三方 LLM API（DashScope），prompt 内容会发送到云端

## 这个系统能替代医生吗？

**不能。** 本系统是 AI 辅助工具，不是医疗设备，不能替代专业医疗诊断。

- AI 分析结果仅供参考，不应作为诊断或治疗的唯一依据
- 如有健康问题，请务必咨询执业医师
- 紧急情况请立即拨打急救电话

## 系统有哪些技术局限？

详见[技术局限](limitations.md)文档。

## 支持哪些语言？

当前主要支持中文（简体）和英文。语音转写目前仅支持英文。

## 需要付费吗？

StarCore MedLink 是开源项目，免费使用。但您需要：
- 自备 LLM API Key（如阿里云百炼 DashScope）
- 自备 Tavily Search API Key（用于联网搜索功能）
- 可选的 ElevenLabs API Key（用于语音功能）
